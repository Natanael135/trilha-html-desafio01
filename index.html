html

A tag html é a raiz de um documento HTML. Ela contém todos os outros elementos HTML da página.

A tag head contém metadados sobre o documento HTML, como o título e links para scripts e estilos.

h1, h2, h3

As tags de título h1 a h6 representam diferentes níveis de cabeçalhos. O h1 é o mais importante e geralmente usado para o título principal, enquanto h2, h3 e outros níveis são usados para subtítulos em ordem de importância.
